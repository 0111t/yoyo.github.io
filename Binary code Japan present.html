How the quetion begin

  • If two tens digits happen to be different in only one digit, then these two digits are called “neighbors”
  • For example: 1234567890 , 1234507890 are two “neighbors”.
  • How many tens digits can be written down so that any two of these numbers are not “neighbors” to each other?

Definition of proper nouns

  • We define this proposition as a n-bit at least d-distance express by A(n,d)
    • 00000 is a 5-bit code
    • the distance between 00000 and 11111 is 5

Basic Theorm 1

  • If d>n then A(n,d) maxium is 1

Proof:

  • To any two binary code the maxium distance between them is the length
  • Thus when d>n you can’t consist any two binary codes with lengnth n which distance is grater than n
  • So A(n,d) {d>n} maxium is 1
